OOP - BIU

Descripción del Proyecto:

Este proyecto consiste en el diseño conceptual de una plataforma de comercio electrónico (e-commerce) utilizando los principios de la Programación Orientada a Objetos (OOP).
El objetivo principal es definir la estructura fundamental del sistema, identificando los requerimientos funcionales (las acciones que los usuarios y administradores podrán
realizar) y los requerimientos no funcionales (las cualidades del sistema como rendimiento, seguridad y usabilidad).
Se implementarán clases en Java, demostrando cómo los objetos interactuarían para ofrecer las funcionalidades definidas, incluyendo la gestión de atributos y comportamientos esenciales.
